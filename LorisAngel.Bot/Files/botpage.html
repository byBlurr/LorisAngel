Lori's Angel

Lori's Angel brings a ton of fun and exciting commands. From funny messages to board games with actual graphics (No more tic-tac-toe using 'x|o|x'). New features are added to Lori's Angel on the regular and anyone can contribute on GitHub!

Permissions

  • SendMessages -- Used to send reponse messages
  • ManageMessages -- Used to delete command messages to keep channels neat and tidy
  • AttachFiles -- Used to send rendered images for games and send profiles
  • KickMembers -- Used by kick command
  • BanMembers -- Used by ban command




Data Usage & Data Requests

Data collected by Lori's Angel is securely stored and only accessed for displaying user profiles, leaderboards and relationship ratings. No data is stored unless directly used by Lori's Angel and data can and will be deleted upon request by a user. To request data to be deleted, message Blurr#9789
